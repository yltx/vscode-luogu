# vscode-luogu

轻松的让你在 VSCode 上使用洛谷。

## 使用说明

请参考 `Github` 上的 [wiki](https://www.github.com/yltx/vscode-luogu/wiki) 页面。

所有功能均有快捷键。

## 完成功能

- 查看题目

- 登录账号

- 注销账号

- 提交代码

- 查看自己测评

- 打卡

- 题目离线查看

- 查看最近一次评测
  
- 查看、发布犇犇

- 查看题解

- 给题解点赞/踩
  
## 开发中功能
  
- 查看、发布讨论

- 比赛相关功能
